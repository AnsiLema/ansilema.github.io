Programme d'extraction d'information de produits

Dans le cadre de ce projet de scraping, j'ai développé des compétences clés en Python et en gestion de projet logiciel. J'ai appris à analyser la structure HTML des pages web pour extraire des données spécifiques grâce à des bibliothèques comme BeautifulSoup. J'ai automatisé le traitement de grandes quantités de données en gérant la pagination et en structurant les résultats dans des fichiers CSV bien formatés. J'ai également implémenté un pipeline ETL complet, incluant le téléchargement et le stockage des images associées. En suivant les bonnes pratiques Git, j'ai structuré un repository propre avec des commits fréquents, un fichier requirements.txt et un .gitignore bien configuré.

Septembre 2024

Logiciel de tournoi d’échecs

Dans ce projet, j’ai développé un logiciel de gestion de tournois d’échecs fonctionnant hors ligne en utilisant la programmation orientée objet (POO) et une architecture Modèle-Vue-Contrôleur (MVC). J'ai conçu des modèles robustes pour les entités clés (joueurs, tournois, matchs), implémenté des contrôleurs pour gérer la logique métier et des vues pour afficher les données via la console. J'ai intégré la persistance des données avec le format JSON, assurant leur sauvegarde et leur récupération. En suivant les bonnes pratiques, notamment la PEP 8, j’ai produit un code propre et maintenable. Enfin, j’ai structuré le projet avec un README clair et un repository bien organisé.

Novembre 2024

Développer la partie front-end d’une application

Dans ce projet, j’ai développé un site web interactif pour visualiser des informations sur les films, comme sur une plateforme de streaming.

J’ai renforcé mes compétences en Front-End en créant une interface conforme aux maquettes avec HTML5, CSS et JavaScript. J’ai utilisé des concepts avancés tels que la programmation asynchrone et les évènements pour charger dynamiquement les données de l’API REST.

La page est entièrement responsive, adaptée aux écrans mobiles, tablettes et ordinateurs, et intègre des fonctionnalités interactives comme des boutons "Voir plus" et des fenêtres modales. Mon code est structuré, documenté et optimisé pour une expérience utilisateur fluide.

Décembre 2024

Développer une Application Web avec Django

Dans ce projet, j’ai développé une application web permettant aux utilisateurs de publier et demander des critiques de livres et d’articles. J’ai utilisé le framework Django pour implémenter le rendu côté serveur, gérer l’authentification des utilisateurs et structurer la base de données conformément au cahier des charges.

J’ai intégré des fonctionnalités avancées comme un système d’inscription et de connexion sécurisé, la gestion des permissions pour contrôler les modifications et suppressions de contenus, ainsi que la pagination et des tags personnalisés pour améliorer l’expérience utilisateur.

L’interface, conforme aux maquettes et aux bonnes pratiques d’accessibilité WCAG, offre une navigation fluide et un rendu professionnel. Mon code est structuré, optimisé et conforme aux normes PEP8, garantissant lisibilité et maintenabilité.
